Instructions for BB Drum Machine

Introduction

BB Drum Machine allows you to easily create a looping drum beat. You merely select which sounds you want to play at the specified times and then start the player to hear your beat. You can edit your beats in real time, load your own sounds into the drum machine, and save your creations for posterity.

BB Drum Machine is broken into the following sections.

Drum Pads

There are nine individual drum pad buttons in this section. Each pad will play the sound currently loaded in it. For keyboard users, if you enable 'Auto-Play Keys' in the Options menu then you can play the sound using the associated keyboard key as well.

There are two sound banks, each with nine sounds that can be loaded into the drum pads. You can switch between the sound banks using the Sounds menu. For keyboard users, you can switch between sound banks using the keyboard combination 'Alt + Number' where Number is the sound bank number you want to load. In order to use this option you must enable 'Alt Shortcuts' in the Options menu.

Volume

The Volume control allows you to control the volume of the sound.

Load

The Load button allows you to load a custom sound into the drum pad. The file you choose must be an audio file and cannot be larger than 500,000 kilobytes.

Edit

The Edit button allows you to change the name of the sound. The maximum number of characters is 40.

Keyboard Shortcuts

While focus is on any drum pad button, you can use the arrow keys to navigate to another drum pad button. Use the Escape key at any time to leave the drum pads section.

Player

The player controls the playing of the drum tracks. Use the Start/Stop button to listen to the drum tracks. The Tempo control determines the speed at which the drum tracks will be played (in Beats per Minute). The Volume control allows you to control the overall volume. For keyboard users, you can start/stop the player using the keyboard combination 'Alt + s'. In order to use this option you must enable 'Alt Shortcuts' in the Options menu.

Keyboard Shortcuts

Use the Escape key at any time to leave the player section.

Drum Tracks

The drum tracks allow you to play a sound at a specified time. Each track represents one measure in 4/4 time, with each button in the track representing a sixteenth note. Activating a button will tell the player to play the sound for that track at that specific beat in the measure.

Similar to the drum pads, there are nine tracks showing, one for each sound currently loaded in the drum pads. You can toggle between the sound banks using the same mechanism as for the drum pads. Even though only 9 tracks are showing, the player will play all 18 drum tracks.

The trash can icon will clear the drum track. If you wish to clear all 18 drum tracks then you can use the 'Clear All Tracks' option under the Options menu.

Keyboard Shortcuts

While focus is on any drum track button, you can use the arrow keys to navigate through the drum tracks. Use the Escape key at any time to leave the drum tracks section.

Menu

Keyboard users can navigate the menu with the arrow keys. The Space Bar or Enter key will select a menu item. Use the Escape key or Tab key at any time to leave the menu. If the menu is in vertical display (for narrower displays) then you will need to activate the menu icon first to expand the menu.

File

Save Session

This allows you to save the current state of the drum machine. It will create a session zip file that you can save to your computer/device.

Load Session

This allows you to load a saved session zip file.

Sounds

This allows you to toggle between the sound banks.

Options

Auto-Play Keys

When activated, allows you to play a sound using the keyboard key specified on the drum pad.

Alt Shortcuts

When activated, allows you to use keyboard shortcuts using Alt key combinations as follows:

  • Alt + Down/Up Arrow: Allows you to move between the different sections of the drum machine.
  • Alt + s: Start/Stop the player
  • Alt + 1 or 2: Switch between sound banks 1 and 2.

Metronome

When activated, the player will include a click sound at the beginning of every quarter note.

Clear All Tracks

Clears all 18 drum tracks.
